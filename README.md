# 🌐 Personal Portfolio Website

A modern, responsive personal portfolio website showcasing my skills, projects, achievements, and professional journey as a **Full Stack Web Developer** and **B.Tech CSE (IoT) student**.

## 🚀 Features

- 👤 Professional introduction and developer profile
- 💼 Project showcase with live demos and GitHub links
- 🛠️ Technical skills categorized by domain
- 🏆 Achievements and certifications
- 📄 Downloadable resume
- 📱 Fully responsive design for desktop, tablet, and mobile
- 🌙 Modern UI with smooth animations and clean user experience
- 📬 Contact section with social media links

## 🛠️ Tech Stack

- **Frontend:** React.js, JavaScript, Tailwind CSS, HTML5, CSS3
- **Icons:** Lucide React, React Icons
- **Build Tool:** Vite
- **Deployment:** Vercel

## 📂 Sections

- Home
- About
- Skills
- Projects
- Education
- Achievements
- Contact

## 📸 Featured Projects

- **DermAI** – AI-powered skin disease detection platform with secure authentication, PDF report generation, and responsive UI.
- **Mockterview** – AI-powered mock interview platform with role-based interview generation, speech recognition, and real-time feedback.

## 📈 Purpose

This portfolio serves as my professional online presence, highlighting my expertise in **Full Stack Web Development**, **AI integration**, and **IoT-based applications** while demonstrating my projects, technical skills, and continuous learning journey.
